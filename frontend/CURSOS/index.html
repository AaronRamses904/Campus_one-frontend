CAMPUS-ONE

TECNOLOGIA

Conoce las bases de la tecnología moderna y cómo impacta nuestra vida diaria. Aprende sobre computadoras, internet, software, hardware, ciberseguridad, inteligencia artificial y más.

MODELO

Aprende a diseñar, analizar y optimizar modelos operativos que orienten la estrategia con la ejecución. Conoce cómo estructurar procesos, roles, tecnología y recursos.

HERRAMIENTAS-TEC

Domina el uso de herramientas digitales clave para el entorno laboral moderno. Aprende a manejar plataformas como Google Workspace, Microsoft 365, Trello, Zoom, Canva y otras aplicaciones esenciales para la productividad.

LENGUAJE

Aprende los fundamentos de la programación utilizando lenguajes. En este curso desarrollarás lógica computacional, resolverás problemas con código y construirás programas.

ALGORITMOS

Aprende a resolver problemas mediante el diseño y análisis de algoritmos. Desarrolla tu lógica de programación, estudia estructuras de control, recursividad y técnicas de optimización.

PÁGINAS WEB

Aprende a crear sitios web desde cero utilizando HTML, CSS y JavaScript. En este curso desarrollarás sitios web modernos, aprenderás los fundamentos, mejorarás interfaces y sentarás las bases del diseño web.
